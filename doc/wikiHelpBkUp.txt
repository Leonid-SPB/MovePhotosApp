===Описание===
Приложение позволяет легко и быстро переносить выбранные фотографии между альбомами пользователя. 

===Как пользоваться===
Для удобства использования рекомендуется добавить приложение в меню слева вашей страницы ВК: щелкните по надписи "''Настройки''" в правом верхнем углу приложения и отметьте пункт "''Выводить ссылку на приложение в меню слева''". 

'''В случае возникновения ошибок''': 
* проверьте, что блокировщики рекламы/скриптов настроены правильно и не мешают работе приложения (не блокирует всплывающие окна и скрипты); 
* попробуйте очистить кэш браузера и перезагрузить приложение (CTRL+F5). 

====Перенос фото между альбомами пользователя====
1. '''Выберите альбом, откуда переносить фотографии.''' Приложение отобразит миниатюры фотографий из выбранного альбома, одновременно отображается до 500 миниатюр. 
Большие альбомы разбиваются на страницы по 500 фоторгафий, для перехода между страницами используйте кнопки со стрелками "&#60;" и "&#62;". 
2. '''Щелчком мыши выберите фото, которые хотите переместить.''' Выбранные фото помечаются красной рамкой. Выбрать все фото на текущей странице альбома можно кнопкой "Все". Выбрать 16 фотографий, показанных на экране можно кнопкой "текущие". 
3. '''Выберите альбом, в который хотите переместить фото.''' 
4. '''Нажмите кнопку "Переместить"''', начнется процесс переноса фото. Ход процесса переноса отображается на шкале слева от кнопки. 

====Перенос фото между альбомами группы====
1. Перенос фото возможен только между альбомами одной и той же группы (т.е. внутри группы). 
2. Требуются права администратора/редактора/модератора группы. 
3. В списке вместо "Мои альбомы", веберите название группы, далее следуйте инструкции "Перенос фото между альбомами пользователя". 

====Сохранение альбома====
''Функция "Сохранение" работает корректно только с браузерами Google Chrome и Mozilla Firefox&#33;'' 
1. '''Выберите альбом, который хотите сохранить.''' Приложение отобразит миниатюры фотографий из выбранного альбома. 
2. '''Щелчком мыши выберите фото, которые хотите сохранить.''' Выбранные фото помечаются красной рамкой. 
3. '''В списке "Куда" выберите "Сохранить".''' 
4. '''Нажмите кнопку "Сохранить"''', откроется страница с фотографиями. В открывшейся странице будут отображаться выбранные фотографии с описаниями и датами. Фотографии выводятся в максимальном доступном качестве. 
5. '''Страницу с фотографиями можно сохранить стандартным способом для вашего браузера''' (например, использовав сочетание клавиш '''CTRL+S'''). Фотографии со страницы обычно сохраняются в отдельную папку в том месте, куда вы сохранили саму страницу. 

''Если страница с фотографиями не открывается, проверьте, что ваш браузер или фаервол не блокирует всплывающие окна.''
