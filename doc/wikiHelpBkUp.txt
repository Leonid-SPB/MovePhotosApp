===Описание===
Приложение позволяет легко и быстро переносить выбранные фотографии между альбомами пользователя. Кроме того, приложение позволяет решить проблему переполнения альбома "Сохраненные фотографии", в который помещается всего лишь 500 фото. 

===Как пользоваться===
Для удобства использования рекомендуется добавить приложение в меню слева вашей страницы ВК: щелкните по надписи "''Настройки''" в правом верхнем углу приложения и отметьте пункт "''Выводить ссылку на приложение в меню слева''". 

'''В случае возникновения ошибок, попробуйте перезагрузить приложение (CTRL+F5). '''

====Перенос фото между альбомами====
1. '''Выберите альбом, откуда переносить фотографии.''' Приложение отобразит миниатюры фотографий из выбранного альбома. 
2. '''Щелчком мыши выберите фото, которые хотите переместить.''' Выбранные фото помечаются красной рамкой. Выбрать все фото сразу можно щелчком мыши по тексту "Снять/выделить все". 
3. '''Выберите альбом, в который хотите переместить фото.''' На размер выбранного альбома также распространяется ограничение в 500 фото. Невозможно перенести фото в переполненный альбом. 
4. '''Нажмите кнопку "Переместить"''', начнется процесс переноса фото. Ход процесса переноса отображается на шкале справа от кнопки. 

====Сохранение альбома====
''Функция "Сохранение" работает корректно только с браузерами Google Chrome и Mozilla Firefox!'' 
1. '''Выберите альбом, который хотите сохранить локально (закачать).''' Приложение отобразит миниатюры фотографий из выбранного альбома. 
2. '''Щелчком мыши выберите фото, которые хотите сохранить.''' Выбранные фото помечаются красной рамкой. Выбрать все фото сразу можно щелчком мыши по тексту "Снять/выделить все". 
3. '''В списке "Куда" выберите "Сохранить локально".''' 
4. '''Нажмите кнопку "Сохранить"''', откроется страница с фотографиями. В открывшейся странице будут отображаться выбранные фотографии с описаниями и датами. Фотографии выводятся в максимальном доступном качестве. 
5. '''Страницу с фотографиями можно сохранить стандартным способом для вашего браузера''' (например, использовав сочетание клавиш '''CTRL+S'''). Фотографии со страницы обычно сохраняются в отдельную папку в том месте, куда вы сохранили саму страницу. 

''Если страница с фотографиями не открывается, проверьте, что ваш браузер или фаервол не блокирует всплывающие окна.''
